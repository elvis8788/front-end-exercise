# Frontend Exercise

Feature:


### 1. Masonry Layout Sorting Algorithm
- Implements a three-column watefall layout sorting algorithm
- Maintains content blocks as close to their original order as possible as
- Fill empty spaces to create a tidy layout

### 2. Infinite Scrolling
- Uses Intersection Observer API
- Automatically loads more content when the button becomes visible

### 3. CSS Layout Update
- Corrects flex-basis calculations when using the gap property
- Maintains proper aspect ratios for all content blocks
- Removes dependency on borders for spacing

### 4. Responsive Design（Bonus）
- Dynamically adjusts the number of columns based on screen width
- Maintains column widths between approximately 100-250px
- Responsive breakpoints for different screen sizes

## Tech
- Vue.js 3
- CSS Grid/Flexbox for layout
- Intersection Observer API

## How to Run
Simply open the index.html file in any modern web browser. No build process or additional setup required.
